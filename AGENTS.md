# Agent notes

- Never write, rewrite, or clean user/editor settings (`settings.json`, `workbench.colorCustomizations`, token customizations, `workbench.colorTheme`).
- Use Umbre-owned generated theme files plus extension global state only.
- Source colors must come from Tailwind CSS; no manual hex literals in source.
- When adding user-facing features, update `README.md` with a minimal, beautiful, human-readable note.
